ข้อความ "Changes not staged for commit" แสดงว่ามีการเปลี่ยนแปลงในไฟล์ `atwtools/src/components/Home.js` 
ที่ไม่ได้ถูกเพิ่มในรายการเตรียม (staging area) สำหรับการ commit ใน local repository ของคุณ.
หากคุณต้องการ commit การเปลี่ยนแปลงในไฟล์ `Home.js`, คุณจะต้องใช้คำสั่ง `git add` 
เพื่อเพิ่มไฟล์นี้เข้าสู่รายการเตรียมก่อนที่คุณจะใช้คำสั่ง `git commit`. นี่คือขั้นตอนที่คุณสามารถทำได้:

1. ใช้คำสั่ง `git add` เพื่อเพิ่มไฟล์ `Home.js` เข้าสู่รายการเตรียม:
   ```
   git add atwtools/src/components/Home.js
   ```
   หรือถ้าคุณต้องการเพิ่มทุกไฟล์ที่มีการเปลี่ยนแปลงในรายการเตรียม:

   ```
   git add .
   ```
2. ใช้คำสั่ง `git commit -m "Your commit message"` เพื่อ commit การเปลี่ยนแปลงที่เพิ่มในรายการเตรียม:

   ```
   git commit -m "Update 1"
   ```
3. หลังจากที่คุณ commit สำเร็จ, คุณสามารถใช้คำสั่ง `git push` เพื่อส่งการ commit ของคุณไปยัง remote repository บน GitHub:

   ```
   git push origin master
   ```
การทำตามขั้นตอนนี้จะทำให้การเปลี่ยนแปลงในไฟล์ `Home.js` ถูกอัปเดตใน remote repository บน GitHub
และสามารถถูกแบ่งปันกับผู้อื่นได้.
